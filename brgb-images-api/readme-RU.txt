Библиотека для обработки изображений в формате BMP и отображения их в ComputerCraft/Tweaked.

ВНЕМАНИЕ: РАБОТАЕТ ТОЛЬКО В ОКРУЖЕНИИ "CraftOS-pc"!!!

Файлы:
	brgb_processing_api — исходный API код
	brgb_processing(console) — консольное приложение для API 
	brgb_viewer — может отображать изображение и видео
	video_convertor — скрипт для конвертации видео в brgb (python)

Основные возможности:
    Чтение BMP файлов — загрузка 24-битных BMP изображений
	Обработка файлов: BRGB, TXT, NFP, BMP для склейки в видео
    Конвертация цветов — преобразование RGB в палитру ComputerCraft (16 цветов)
    Дизеринг Флойда-Стейнберга — улучшение качества преобразования цветов
    переписанный формат BRGB — теперь поддерживает видео поток

Ключевые функции:
    полностью описано в справке в консольном приложении

(в коде могу содержатся неизвестные ошибки, дайте знать если вы их обнаружите)
в скоре передам код для конвертации mp4 и прочих видео файлов
