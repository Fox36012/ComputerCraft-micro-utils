Библиотека для обработки изображений в формате BMP и отображения их в ComputerCraft/Tweaked.

ВНЕМАНИЕ: РАБОТАЕТ ТОЛЬКО В ОКРУЖЕНИИ CraftOS-pc!!!

Файлы:
	brgb_processing_api — исходный код
	brgb_processing(console) — консольная версия (Использование: brgb_process <bmp_file> <brgb_file> <useDithering>)
	brgb_processing_api(optim) — оптимизированная версия
	brgb_viewer — может только отображать изображение (Использование: brgb_viewer <path_to_brgb_file>)

Основные возможности:
    Чтение BMP файлов - загрузка 24-битных BMP изображений
    Конвертация цветов - преобразование RGB в палитру ComputerCraft (16 цветов)
    Дизеринг Флойда-Стейнберга - улучшение качества преобразования цветов
    Собственный формат BRGB - компактное хранение изображений
    Отображение изображений - вывод на экран через графический режим

Ключевые функции:
    processBMP() - конвертирует BMP в BRGB с дизерингом
    displayBRGB() - отображает BRGB файл на экране
    previewImage() - предпросмотр изображения в графическом режиме
    loadBRGB()/saveBRGB() - работа с собственным форматом

Особенности:
    Поддержка алгоритма дизеринга для сглаживания цветовых переходов
    Автоматическое определение ближайших цветов из палитры CC
    Эффективное хранение данных (1 байт на пиксель)
    Обработка ошибок чтения файлов

(в коде могу содержатся неизвестные ошибки, дайте знать если вы их обнаружите)
