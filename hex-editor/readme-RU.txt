Шестнадцатеричный редактор для просмотра и редактирования бинарных файлов в ComputerCraft/Tweaked.

работает только с уже существующими файлами (в будущем это будет исправленно)
Основные возможности:
    Просмотр файлов - отображение в hex и ASCII формате
    Редактирование - пошаговое изменение байтов через hex-ввод
    Навигация - стрелками по ячейкам, скроллинг больших файлов
    Сохранение - автоматическое сохранение изменений (Tab)

Интерфейс:
    Левая панель - offset (смещение в файле)
    Центральная панель - hex-представление байтов (16 колонок)
    Правая панель - ASCII-представление символов
    Курсор - подсветка текущей редактируемой ячейки

Управление:
    Стрелки - перемещение курсора
    0-9, A-F - ввод hex-значений
    Tab - сохранить файл
    Escape - выход из редактора

Особенности:
    Автоматическое определение размера терминала
    Поддержка файлов любого размера с виртуальной прокруткой
    Защита от выхода за границы файла
    Валидация вводимых hex-значений

(в коде могу содержатся неизвестные ошибки, дайте знать если вы их обнаружите)
