Документация API для descutils.lua

descutils.lua - это библиотека для создания и управления графическими кнопками в среде ComputerCraft/Tweaked. Библиотека предоставляет простой API для создания интерактивных элементов пользовательского интерфейса.
Основные функции:

descutils.createButton(x, y, width, height)

Создает новую кнопку и возвращает объект кнопки.

Параметры:
    x, y - координаты верхнего левого угла кнопки
    width, height - размеры кнопки

Возвращает: объект кнопки

Пример
lua:{
	local button = descutils.createButton(2, 3, 10, 3)
}
--------------------------------------------
Методы объекта кнопки:

button:setColor(newColor)

Устанавливает цвет фона кнопки.

Параметры:
    newColor - цвет (число или таблица цветов из CC)

Возвращает: сам объект кнопки (для цепочки вызовов)
--------------------------------------------
button:setTextColor(newColor)

Устанавливает цвет текста кнопки.

Параметры:
    newColor - цвет (число или таблица цветов из CC)

Возвращает: сам объект кнопки
--------------------------------------------
button:setLabel(alignment, text)

Устанавливает текст кнопки и его выравнивание.

Параметры:
    alignment - строка выравнивания (формат: "ВГ", где В - вертикальное, Г - горизонтальное)
    text - текст кнопки

Форматы выравнивания:
    Вертикальное: "U" (верх), "C" (центр), "D" (низ)
    Горизонтальное: "L" (левый), "C" (центр), "R" (правый)
    Примеры: "UL", "CC", "DR"

Возвращает: сам объект кнопки
--------------------------------------------
button:setName(newName)

Устанавливает уникальное имя кнопки для идентификации.

Параметры:
    newName - строковое имя кнопки

Возвращает: сам объект кнопки
--------------------------------------------
button:setVisible(isVisible)

Показывает или скрывает кнопку.

Параметры:
    isVisible - булево значение видимости

Возвращает: сам объект кнопки
--------------------------------------------
button:setEnabled(isEnabled)

Включает или отключает кнопку.

Параметры:
    isEnabled - булево значение активности

Возвращает: сам объект кнопки
--------------------------------------------
button:draw()

Отрисовывает кнопку на экране.
button:contains(clickX, clickY)

Проверяет, находится ли точка внутри кнопки.

Параметры:

    clickX, clickY - координаты точки

Возвращает: true если точка внутри кнопки, иначе false
--------------------------------------------
Глобальные функции библиотеки:

descutils.isClicked(clickX, clickY)

Проверяет клик по любой из созданных кнопок.
Параметры:
    clickX, clickY - координаты клика

Возвращает:
    name - имя кнопки, по которой кликнули (или nil)
    true/false - был ли клик по какой-либо кнопке
--------------------------------------------
descutils.getButton(name)

Возвращает кнопку по имени.

Параметры:
    name - имя кнопки
	
Возвращает: объект кнопки или nil
--------------------------------------------
descutils.removeButton(name)

Удаляет кнопку по имени.

Параметры:
    name - имя кнопки для удаления
--------------------------------------------
descutils.drawAll()

Отрисовывает все созданные кнопки.
--------------------------------------------
descutils.clearAll()

Удаляет все кнопки.
descutils.getButtonCount()

Возвращает количество созданных кнопок.

Возвращает: число кнопок
--------------------------------------------
Пример использования
lua:{
	local descutils = require("descutils")
	
	-- Создание кнопки
	local myButton = descutils.createButton(5, 5, 15, 3)
		:setColor(colors.blue)
		:setTextColor(colors.white)
		:setLabel("CC", "Нажми меня!")
		:setName("my_button")
		:setEnabled(true)
	
	-- Отрисовка всех кнопок
	descutils.drawAll()
	
	-- Обработка кликов
	local event, button, x, y = os.pullEvent("mouse_click")
	local buttonName, clicked = descutils.isClicked(x, y)

	if clicked then
		print("click on button: " .. buttonName)
	end
}
Примечания:
    Кнопки автоматически обрезают длинный текст (добавляется "...")
    Отключенные кнопки отображаются серым цветом
    Все методы кнопки поддерживают цепочку вызовов (chaining)
    Координаты и размеры используют систему ComputerCraft (1-based)
