Файловый менеджер для ComputerCraft/Tweaked с графическим интерфейсом.
Основные возможности:

    Навигация по файловой системе - просмотр папок и файлов

    Управление файлами:
        Выполнение программ (.lua и другие)
        Редактирование файлов
        Удаление с подтверждением

    Информация о файлах:
        Размер файлов (автоматическое форматирование)
        Статус "только для чтения"
        Тип (папка/файл)

    Интерфейс:
        Прокрутка длинных списков
        Контекстное меню действий
        Подтверждение удаления
        Справка (F1)

Управление:
    Стрелки - навигация по списку
    Enter - открыть папку или меню действий
    Delete - удалить файл
    Backspace - на уровень вверх
    Home - в корневую папку
    Tab - выход из программы

Особенности:
    Автоматическое определение размера терминала
    Поддержка длинных имен файлов (обрезка с "...")
    Визуальные индикаторы для папок и файлов
    Защита от удаления системных файлов (read-only)
