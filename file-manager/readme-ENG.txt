File manager for ComputerCraft/Tweaked with graphical interface.
Main Features:
    File system navigation - browse folders and files

    File management:
        Execute programs (.lua and others)
        Edit files
        Delete with confirmation

    File information:
        File size (auto-formatted)
        Read-only status
        Type (folder/file)

    Interface:
        Scrolling for long lists
        Context action menu
        Delete confirmation
        Help (F1)

Controls:
    Arrow keys - navigate list
    Enter - open folder or action menu
    Delete - delete file
    Backspace - go up one level
    Home - go to root folder
    Tab - exit program

Features:
    Automatic terminal size detection
    Long filename support (truncated with "...")
    Visual indicators for folders and files
    Protection against deleting system files (read-only)
